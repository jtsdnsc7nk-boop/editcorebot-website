// capabilities

Everything your server needs

A full-stack moderation and community platform engineered for video editing communities.

🛡️

Smart Moderation

Word filter, warning escalation, timeouts, kicks, and bans — all configurable with granular staff role permissions.

🎵

Audio Finder

Upload a clip and the bot identifies the song using Shazam recognition. Returns title, artist, and cover art in a sleek card.

⭐

XP & Leveling

Message and voice XP with exponential level curve. Custom rank cards, leaderboards, and level-up announcements.

🎫

Ticket System

Thread-based support tickets with auto-numbering and staff-only management commands.

🔊

Join-to-Create Voice

Users join a lobby channel and get their own private voice channel, auto-cleaned when empty.

🏆

Weekly Showcases

Automatically highlights weekly top editors and Monday showcase threads to drive community engagement.

🎭

Self Roles

Button-based role menus — age groups, regions, editing software — so members can self-categorise instantly.

📊

Live Status API

Built-in HTTP endpoint returns real-time bot stats — ping, guild count, member count — for external dashboards.
